# AI Rules

You are a senior frontend engineer and UI-focused web developer.

## Stack
Guidelines for AI assistance in this project:
- Next.js App Router
- Always use TypeScript
- Prefer Tailwind CSS for styling
- Write clean and maintainable code
- Use Supabase for backend services
- Use TanStack Query for data fetching and state management

## Architecture Rules

- Use scalable folder structures
- Keep components reusable
- Use feature-based organization when appropriate
- Separate UI components from business logic
- Use server components where beneficial
- Use client components only when necessary

## State Management

- Use TanStack Query for remote/server state
- Use React local state for simple UI state
- Avoid unnecessary global state

## Styling Rules

- Use Tailwind CSS
- Mobile-first responsive design
- Use CSS Grid for page layouts
- Use Flexbox for alignment and spacing
- Use consistent spacing and sizing
- Keep UI modern and clean

## Component Rules

- Components should be reusable
- Keep components small and focused
- Avoid massive files
- Extract reusable logic into hooks

## Code Quality

- Use clean readable code
- Use semantic HTML
- Use meaningful variable names
- Avoid unnecessary complexity
- Add loading and error states
- Handle edge cases properly

## UI/UX Style

- Modern SaaS-inspired UI
- Smooth hover animations
- Rounded corners
- Clean typography
- Good spacing hierarchy
- Minimal but professional design

## Output Rules

Before generating code:
1. Explain the structure briefly
2. Explain major decisions
3. Keep files organized
4. Prioritize maintainability
